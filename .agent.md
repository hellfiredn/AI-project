# Agent Overview - AI-project

Cap nhat gan nhat: 2026-05-14

## Tong quan

Day la workspace gom nhieu nhom cong viec lien quan den WordPress, affiliate/deal, san xuat video AI va tu dong hoa fanpage. Khong phai mot app don le. Khi lam viec, hay xac dinh dung khu vuc truoc khi sua.

Nhanh hieu du an:

- `deal/`: ban cai WordPress chinh cho website affiliate "Tap Hoa Giam Gia". Phan tu viet quan trong nam trong theme `deal/wp-content/themes/codedeal/` va plugin `deal/wp-content/plugins/codedeal-accesstrade/`.
- `webwp/`: ban cai WordPress khac, theme giao dien giao duc/online learning tu Figma, nam trong `webwp/wp-content/themes/webwp-theme/`.
- `remotion-story/`: project Remotion/React/TypeScript de render video doc 1080x1920 tu 6 canh, voiceover va nhac nen.
- `social-media-fanpage/`: script PowerShell va tai lieu de tao/dang bai Facebook fanpage, co lich noi dung Excel.
- `outputs/`: media da render, audio/video/preview. Thuong la output nang, khong nen sua neu task khong lien quan.
- `prompt/`, `huong-dan-pipeline-ai-video.md`, `script-mau-ke-chuyen.md`: prompt va tai lieu quy trinh san xuat noi dung/video.

## Nguyen tac cho agent

- Uu tien sua code tu viet, tranh sua core WordPress trong `wp-admin/`, `wp-includes/` hoac plugin vendor neu khong co yeu cau ro.
- Khong commit hoac in noi dung file local co token/secret, dac biet `social-media-fanpage/facebook-env.local.ps1` va cac file oauth/local state.
- Thu muc `node_modules/`, `logs/`, `outbox/`, `outputs/` la phu tro/output; chi dung lam dau vao/tham chieu khi can.
- Repo dang co thay doi chua commit. Khong revert file do user/agent khac sua tru khi duoc yeu cau.
- Neu doc file tieng Viet bi loi font tren terminal PowerShell, thu dat encoding UTF-8 truoc khi doc.

## WordPress affiliate: `deal/`

`deal/` la WordPress install chay duoi XAMPP. Phan custom chinh:

- Theme: `deal/wp-content/themes/codedeal/`
- Plugin sync affiliate: `deal/wp-content/plugins/codedeal-accesstrade/`
- Plugin vendor dang co: ACF Pro, Rank Math SEO, Contact Form 7, Duplicate Page, WPCode/Insert Headers and Footers.

### Theme `codedeal`

Theme nay dung cho website tong hop deal, coupon va meo mua sam.

File quan trong:

- `functions.php`: bootstrap theme, enqueue CSS/JS, helper format gia, tinh giam gia, lay store, banner trang chu.
- `inc/post-types.php`: dang ky CPT `deal`, `coupon` va taxonomy `store`.
- `inc/metaboxes.php`: metabox thong tin deal/coupon/store color.
- `inc/acf-fields.php`: ACF field group cho banner slider trang chu.
- `inc/template-tags.php`: render card deal, coupon, store logo, blog card.
- `page-trang-chu.php`: template trang chu, gom hero, banner slider, flash deal, top ban chay, deal noi bat, coupon, blog.
- `assets/css/main.css`: style chinh.
- `assets/js/main.js`: JS giao dien, copy coupon, slider.
- `archive-deal.php`, `archive-coupon.php`, `single-deal.php`, `single-coupon.php`, `taxonomy-store.php`: trang listing/detail.

Data model chinh:

- `deal`: title, content, excerpt, thumbnail, taxonomy `store`, meta `_cd_price_old`, `_cd_price_new`, `_cd_affiliate_url`, `_cd_expires_at`, `_cd_image_url`, `_cd_featured`.
- `coupon`: title, content, taxonomy `store`, meta `_cd_code`, `_cd_affiliate_url`, `_cd_expires_at`, `_cd_min_order`.
- `store`: taxonomy cho san/cua hang, co term meta `store_color`.

### Plugin `codedeal-accesstrade`

Plugin dong bo Accesstrade Vietnam vao CPT cua theme `codedeal`.

File quan trong:

- `codedeal-accesstrade.php`: entrypoint plugin, load class, hook activate/deactivate, WP-CLI command `codedeal-at`.
- `includes/class-api.php`: wrapper Accesstrade API (`campaigns`, `promotions`, `top_products`, `datafeeds`, `orders`, `deeplink`, `probe`).
- `includes/class-sync.php`: dong bo campaign, coupon, top product, datafeed, performance/orders.
- `includes/class-mapper.php`: map/upsert API rows vao `deal`, `coupon`, `store`.
- `includes/class-cron.php`: lich sync/generate.
- `includes/class-generator.php`: tao bai tong hop/top discount/flash sale/coupon roundup.
- `includes/class-admin.php`: UI admin plugin.
- `includes/class-cli.php`: lenh WP-CLI.

Lenh WP-CLI neu moi truong co `wp`:

```powershell
cd D:\xampp\htdocs\AI-project\deal
wp codedeal-at diagnose
wp codedeal-at sync all
wp codedeal-at sync top
wp codedeal-at sync datafeed
wp codedeal-at campaigns
wp codedeal-at logs
```

## Remotion video: `remotion-story/`

Project Remotion de render video ke chuyen doc 1080x1920, 60s.

Tech:

- Remotion `4.0.460`
- React `19.2.3`
- TypeScript
- Tailwind v4 config qua `@remotion/tailwind-v4`
- Font Be Vietnam Pro tu `@remotion/google-fonts`

File quan trong:

- `src/scenes.ts`: cau hinh FPS, kich thuoc, duration, audio source, danh sach 6 scene va caption.
- `src/Composition.tsx`: ghep cac scene, voiceover va music.
- `src/Scene.tsx`: render media, Ken Burns zoom, fade, caption overlay.
- `src/Root.tsx`: dang ky composition `Story`.
- `remotion.config.ts`: config render.
- `public/`: dat `scene1.mp4` den `scene6.mp4`, `narration.mp3`, `music.mp3`.

Lenh hay dung:

```powershell
cd D:\xampp\htdocs\AI-project\remotion-story
npm run dev
npm run lint
npx remotion render Story out/story.mp4
```

Khi thay doi kich thuoc/duration/caption, sua `src/scenes.ts`. Khi thay doi animation/caption layout, sua `src/Scene.tsx`.

## Python video pipeline

Root co hai script render video bang Python/Pillow/NumPy/ffmpeg:

- `render_cham_ngon_video.py`: tao video 30s phong canh lang que, tao music bed, thu sinh voice bang `edge_tts`, render frame bang Pillow/NumPy, mux bang ffmpeg.
- `render_animated_3d_background_video.py`: dung anh nguon `outputs/cham-ngon-song-hay/lang-que-3d-source.png`, animate camera/wind/smoke/text, mux voice/music.

Thu vien can co:

- `edge_tts`
- `imageio_ffmpeg`
- `numpy`
- `Pillow`

Lenh chay:

```powershell
cd D:\xampp\htdocs\AI-project
python render_cham_ngon_video.py
python render_animated_3d_background_video.py
```

Output mac dinh nam trong `outputs/cham-ngon-song-hay/`.

## Fanpage automation: `social-media-fanpage/`

Thu muc nay ho tro content calendar va dang bai Facebook Page.

File quan trong:

- `content-calendar.xlsx`: lich noi dung, topic mix, cadence, KPI.
- `AUTO_POST_PROMPT.md`: workflow cho automation tao bai hang ngay.
- `huong-dan-tu-dong-dang-bai.md`: huong dan len lich bang Meta Business Suite.
- `publish-facebook-page-post.ps1`: dang text post, tuy chon link va first comment.
- `publish-facebook-page-photo.ps1`: dang photo post.
- `facebook-env.example.ps1`: mau bien moi truong.
- `facebook-env.local.ps1`: file local chua token, khong doc/in/commit.
- `logs/`: log NDJSON khi dang/dry-run.

Bien moi truong can khi publish:

- `FACEBOOK_PAGE_ID`
- `FACEBOOK_PAGE_ACCESS_TOKEN`
- `FACEBOOK_GRAPH_API_VERSION`
- `FACEBOOK_DRY_RUN`

Khuyen nghi giu `FACEBOOK_DRY_RUN=1` khi test.

Lenh mau:

```powershell
. .\social-media-fanpage\facebook-env.local.ps1
.\social-media-fanpage\publish-facebook-page-post.ps1 -MessageFile .\social-media-fanpage\outbox\YYYY-MM-DD-facebook-caption.txt -DryRun
.\social-media-fanpage\publish-facebook-page-photo.ps1 -ImageFile .\social-media-fanpage\poster-yen-chung-facebook.png -MessageFile .\social-media-fanpage\post-yen-chung-caption.txt -DryRun
```
